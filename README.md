# Krptkn

Krptkn is a spider, metadata extractor, metadata analyzer and report generator.

# Roadmap

- [ ] Spider

- [ ] Metadata analyzer
- [x] Metadata filter
- [ ] Report generator
- [ ] Meta
  - [x] Create tests
  - [x] Continuous integration
  - [x] Module documentation
  - [ ] Function documentation
  - [ ] Schematics
